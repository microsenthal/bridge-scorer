⚙ First-time Firebase Setup

This app needs a free Firebase database to sync scores across devices. Follow these steps:

  1. Go to console.firebase.google.com
  2. Create a project (any name, disable Analytics)
  3. Click Firestore Database → Create Database (production mode is fine)
  4. Go to Project Settings → Your apps → Add app → Web
  5. Copy the firebaseConfig object and paste it below
  6. In Firestore, go to Rules and set: allow read, write: if true; then publish
